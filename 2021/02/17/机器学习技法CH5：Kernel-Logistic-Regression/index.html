CH5пјҡKernel Logistic Regression

Soft-Margin SVM as Regularzied

еҪ“$(x_n,y_n)$и¶ҠиҝҮиҫ№з•Ңж—¶пјҢй”ҷиҜҜе°ұжҳҜ$1-y_n(w^Tz_n+b)$,еҪ“$(x_n,y_n)$жІЎжңүи¶ҠиҝҮиҫ№з•Ңж—¶пјҢиҜҙжҳҺд»–жҳҜжӯЈзЎ®зҡ„жІЎжңүй”ҷиҜҜпјҢеҚі$\xi=0$,йӮЈд№ҲжҲ‘д»¬з»јдёҠжүҖиҝ°пјҡеҸҜд»ҘжҠҠ$\xi$жҚўжҲҗеҸҰдёҖз§ҚеҶҷжі•пјҡ$max(1-y_n(w^Tz_n+b),0)$

жӯӨж—¶жҲ‘д»¬зҡ„SVMеҸҜд»Ҙиҝҷд№ҲеҶҷ:

иҝҷж ·дёҖжқҘпјҢжҲ‘д»¬жҠҠд№ӢеүҚжҠҪиұЎзҡ„$\xi$жҰӮеҝөиҪ¬еҢ–дёәдәҶе…·дҪ“зҡ„ејҸеӯҗпјҢеҗҢж—¶жҲ‘д»¬зҺ°еңЁеҸҜд»ҘйҖҡиҝҮ $b,w$зҡ„дёҚеҗҢеҸ–еҖјжңҖе°ҸеҢ–дёӢејҸзҡ„soft-margin SVMй—®йўҳпјҡ

еӣ жӯӨsoft-margin SVMе°ұжҳҜдёҖдёӘL2 regularizationзҡ„й—®йўҳгҖӮ

д»–д»¬д№Ӣй—ҙжңүдёҖдәӣзӣёдә’зҡ„е…ізі»пјҡ

SVM versus Logistic Regression

йҰ–е…ҲжҲ‘д»¬еҜ№жҜ”дёҖдёӢ0-1error е’Ң SVMдёӯзҡ„error measureзҡ„еҢәеҲ«пјҡ

жҲ‘д»¬дёҚйҡҫзңӢеҮәпјҡSVMдёӯзҡ„error measureжҖ»жҳҜеӨ§дәҺ0-1error measureгҖӮ

жҲ‘д»¬еңЁlogistic regressionдёӯз”ЁеҲ°зҡ„error measureеҸ«еҒҡпјҡscaled cross-entropy error пјҢиҜҰз»ҶжҺЁеҜји§Ғпјҡlogistic regressionгҖӮ

йҰ–е…ҲжҲ‘д»¬жңүдёӘжҰӮеҝөеҸ«еҒҡпјҡcross entropy

дҪҶжҳҜдёәдәҶеӣҫеғҸеҜ№жҜ”пјҢж”№иҝӣжҲҗдәҶscaled cross-entropy,еҚі$ln$жҚўжҲҗдәҶ$log_2$

жҲ‘д»¬еҜ№жҜ”дёҖдёӢдёүиҖ…пјҡ

иҝҷж ·зңӢжқҘпјҡ SVMзҡ„error measureж–№жі•е’Ңlogistic regressionзҡ„error measureж–№жі•еҫҲзӣёдјјгҖӮ

SVMжӯӨж—¶еҸҜд»ҘзңӢдҪң: жңүзқҖL2-regularizedзҡ„logistic regression

еӣ жӯӨзҺ°еңЁжҲ‘д»¬жңүдәҶдёҖдёӘж–°зҡ„жғіжі•пјҡ

еҪ“жҲ‘д»¬жұӮеҮәSVMзҡ„и§Јзҡ„ж—¶еҖҷпјҢжҲ‘д»¬жҳҜеҗҰеҸҜд»ҘйҖҡиҝҮSVMзҡ„и§ЈжқҘеҸҚжҳ logistics regressionйҮҢзҡ„еҮ зҺҮй—®йўҳе‘ўпјҹ

SVM for Soft Binary

  • жҲ‘д»¬и·‘soft-margin SVMз„¶еҗҺеҫ—еҲ°$bпјҢw$, з„¶еҗҺдј еӣһеҲ°g(x)еҪ“дҪңеҲҶзұ»еҷЁгҖӮиҝҷж ·зҡ„ж–№жі•иЎЁзҺ°еҫ—дёҖиҲ¬дёҚй”ҷпјҢдҪҶжҳҜе°‘дәҶдёҖдәӣйҖ»иҫ‘еӣһеҪ’зҡ„зү№зӮ№пјҢlogistics regressionдёӯзҡ„жўҜеәҰдёӢйҷҚи·‘еҮәжқҘзҡ„з»“жһңејәи°ғзҡ„жҳҜmaximum(жңҖеӨ§)пјҢдҪҶжҳҜзҺ°еңЁSVMи·‘еҮәжқҘзҡ„иӮҜе®ҡе’ҢжңҖеӨ§жңүдёҖзӮ№зӮ№зҡ„е·®и·қгҖӮ

  • жҲ‘д»¬иҝҳжңүдёҖз§Қжғіжі•е°ұжҳҜпјҡ жҲ‘д»¬еҒҡйҖ»иҫ‘еӣһеҪ’ж—¶йңҖиҰҒйҖүдёҖдёӘиө·е§ӢзӮ№з„¶еҗҺеҺ»еҒҡSGD/GD(жўҜеәҰдёӢйҷҚ)пјҢйӮЈд№ҲжҲ‘д»¬еҸҜд»ҘжҠҠSVMжұӮеҮәжқҘзҡ„з»“жһңеҪ“дҪңиҝӯд»Јзҡ„иө·е§ӢзӮ№пјҢз„¶еҗҺеҶҚеҺ»еҒҡSGD/GD. дҪҶжҳҜиҝҷж ·зҡ„еҒҡжі•дјҡдҪҝеҫ—жҲ‘д»¬SVMдёӯзҡ„kernelзӯүзү№зӮ№жІЎжі•з”ЁеҲ°гҖӮ

жҲ‘д»¬жғізқҖиЎҘе…ЁдёӨиҖ…зҡ„зјәзӮ№гҖӮеӣ жӯӨжҲ‘д»¬ж·»еҠ дёҠдёӨдёӘиҮӘз”ұеәҰA,B

  • з”ЁA,BеҺ»еҫ®и°ғhyperplane дҪҝеҫ— е…¶ж»Ўи¶іmaximumпјҢеҚіlogistics regressionзҡ„зү№зӮ№

  • еҗҢж—¶жҲ‘д»¬з”ұдәҺдҝқз•ҷдәҶ$\phi$,жүҖд»Ҙkernelзҡ„зү№зӮ№д№ҹдҝқз•ҷдәҶдёӢжқҘгҖӮ

еҗҢж—¶жҲ‘д»¬д№ҹиҰҒжіЁж„ҸеҲ°пјҡ

AдёҖиҲ¬еӨ§дәҺ0пјҢеҰӮжһңдҪ зҡ„Aе°ҸдәҺ0пјҢиҜҙжҳҺдҪ зҡ„$w_{SVM}$жұӮеҫ—жңүй—®йўҳгҖӮBдёҖиҲ¬зәҰзӯүдәҺ0пјҢд»–еҸӘжҳҜе№із§»hyperplaneпјҢз”ұдәҺжҲ‘д»¬SVMжұӮеҮәзҡ„е·Із»ҸжҜ”иҫғеҮҶзЎ®дәҶпјҢжүҖд»ҘдёҖиҲ¬дёҚдјҡжңүд»Җд№ҲеӨӘеӨ§зҡ„еҠЁдҪңгҖӮ

йӮЈжҲ‘д»¬ж–°зҡ„еӣһеҪ’й—®йўҳеҸҜд»ҘеҶҷдҪңпјҡ

жҲ‘д»¬и§ӮеҜҹиҝҷдёӘејҸеӯҗпјҡ

иҝҷдёӘж“ҚдҪңе°ҶдёҖдёӘеӨҡз»ҙзҡ„ж•°жҚ®$x_n$иҪ¬жҚўдёәдәҶдёҖдёӘе…·дҪ“зҡ„ж•°:

$\phi_{SVM}(x_n)=w_{SVM}^T\phi(x_n)+b_{SVM}$

жҲ‘д»¬еҶҚз”ЁиҝҷдёӘж•°жқҘз”ЁA,BеҒҡи°ғж•ҙеҚіеҸҜпјҢжӯӨж—¶logistics regressionжӯӨж—¶еҸӘйңҖиҰҒеҒҡA,BдёӨдёӘз»ҙеәҰеҚіеҸҜгҖӮ

дҪҶжҳҜиҝҷж ·еҫ—еҲ°зҡ„дёҚжҳҜZз©әй—ҙзҡ„жңҖдјҳи§ЈпјҢеҸӘжҳҜдёҖдёӘз»ҸиҝҮA,Bи°ғж•ҙеҗҺжҜ”иҫғеҘҪзҡ„и§ЈгҖӮ

Kernel Logistic Regression

жңҖдјҳи§Јзҡ„$w$дёӯд№ҹеҢ…еҗ«ж ёжҠҖе·§пјҢе№¶дё”д№ҹжҳҜ$z_n$зҡ„зәҝжҖ§з»„еҗҲ

е…¶е®һж— и®әжҳҜSVMпјҢPLAиҝҳжҳҜlogReg by SGDйғҪжҳҜ$z_n$зҡ„зәҝжҖ§з»„еҗҲпјҡ

жүҖд»ҘжҲ‘д»¬еҫ—еҮәз»“и®әпјҡ

вҖӢ жҲ‘д»¬зҡ„жңҖдҪізҡ„$w_*$дёҖе®ҡжҳҜз”ұдёӨйғЁеҲҶз»„жҲҗзҡ„пјҢдёҖйғЁеҲҶжҳҜе№іиЎҢдәҺz-з©әй—ҙеҗ‘йҮҸ$z_n$д»¬жүҖеј жҲҗзҡ„з©әй—ҙпјҢеҸҰдёҖйғЁеҲҶжҳҜеһӮзӣҙдәҺиҝҷдёӘеј жҲҗзҡ„з©әй—ҙгҖӮ

вҖӢ жҲ‘д»¬иӮҜе®ҡеёҢжңӣеһӮзӣҙдәҺеј жҲҗзҡ„з©әй—ҙзҡ„wйғЁеҲҶпјҢеҚі$w_вҠҘ=0$.

еҰӮжһң$w_вҠҘ\ne 0$дјҡеҫ—еҲ°д»Җд№Ҳз»“жһңе‘ўпјҹжҲ‘д»¬иҖғиҷ‘дёҖдёӢдёӢйқўдёӨдёӘйғЁеҲҶпјҡ

жҲ‘д»¬еҸ‘зҺ°иҝҷйғЁеҲҶе№¶дёҚдјҡиў«еҪұе“Қеӣ дёә$w_вҠҘz_n=0$жҳҜдёҖдёӘдәӢе®һпјҢеһӮзӣҙдәҺ$z_n$зҡ„еҗ‘йҮҸе’Ң$z_n$зӣёд№ҳеҪ“з„¶зӯүдәҺ0гҖӮ

з¬¬дәҢйғЁеҲҶпјҢеҪ“жҲ‘д»¬иҖғиҷ‘еӨ§е°Ҹж—¶е°ұеҮәзҺ°дәҶй—®йўҳпјҢеҸҜд»ҘзңӢеҮәеҗ‘йҮҸдјҡеӨ§дёҖдәӣгҖӮ

дҪҶжҳҜиҝҷе°ұеҮәзҺ°дәҶй—®йўҳпјҢиҝҷжҸҗзӨәжҲ‘д»¬$w_^Tw_>w_вҲҘ^Tw_вҲҘ$,иҝҷиҜҙжҳҺжҲ‘д»¬зҡ„жңҖдјҳи§ЈдёҚжҳҜжңҖдјҳи§ЈдәҶз«ҹз„¶пјҢиҝҷж„Ҹе‘ізқҖжҲ‘д»¬зҡ„$w_вҠҘ=0$жҳҜжҒ’жҲҗз«Ӣзҡ„пјҢдёҠйқўзӣёеҪ“дәҺжҲ‘д»¬еңЁеҸҚиҜҒиҝҷдёӘз»“и®әгҖӮ

$w_вҠҘ=0$жҳҜжҒ’жҲҗз«Ӣзҡ„е°ұж„Ҹе‘ізқҖпјҡжҲ‘д»¬зҡ„$w_*$дёҖе®ҡжҳҜз”ұ$z_n$зҡ„зәҝжҖ§з»„еҗҲеҫ—жқҘзҡ„гҖӮ

иҝҷж ·дёҖжқҘжҲ‘д»¬е°ұеҸҜд»ҘеҠ дёҠж ёжҠҖе·§дәҶгҖӮ

жӯӨж—¶жҲ‘д»¬жұӮи§Ј$\beta$еҚіеҸҜгҖӮ

жҖҺд№ҲжұӮе‘ўпјҹ GD/SGDйғҪжҳҜеҫҲеҘҪзҡ„йҖүжӢ©гҖӮ

еҸҰдёҖз§Қи§Ҷи§’пјҡ

  • зӣёеҪ“дәҺжқғйҮҚ$\beta$е’ҢеҒҡfeature transformеҗҺзҡ„ж•°жҚ®зҡ„д№ҳз§ҜгҖӮ

  • иҝҷйғЁеҲҶеҸҜд»ҘзңӢдҪңдёҖз§Қзү№ж®Ҡзҡ„жӯЈеҲҷеҢ–гҖӮ

  • еӣ жӯӨKernel Logistic RegressionеҸҜд»ҘзңӢдҪң$\beta$зҡ„зәҝжҖ§жЁЎеһӢпјҢеҸӘдёҚиҝҮе®ғжҳҜйҖҡиҝҮkernel transformеҗҺзҡ„ж•°жҚ®е’ҢдёҖз§Қзү№ж®Ҡзҡ„kernel regularizer еҫ—еҲ°зҡ„з»“жһңгҖӮ
